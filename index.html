Research

Machine-level: I am interested in developing highly efficient realtime robot algorithms (using C++ and Python mostly) that can run with limited resources (e.g., edge computers on robots such Nvidia Orin NX).

"Intelligence"-level: I am passionate about developing "intelligent" multi-robot systems that can achieve more demanding tasks such as cooperative logistics, manipulation, and more based on efficient realtime programs.

Specifically, motion planning, SLAM, optimal control, and machine learning for robots are also my interested fields.
